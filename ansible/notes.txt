ansible playbooks
-----------------
first, add ssh key:

    ssh-add ~/.ssh/id_rsa

to run a single playbook:

    ansible-playbook -i hosts server-setup-common.yml --ask-become-pass

check playbook syntax (does this work?!):

    ansible-playbook --syntax-check playbook.yml
